# GoFileCLI (gfcli)
Quickly upload and download with GoFile.io.

## Requirements
- `curl` & `jq` must be installed on your system.

## Features
- Upload Files CLI
- Download Files CLI

## Installation
simply run:
```bash
sudo wget https://raw.githubusercontent.com/officialputuid/GoFileCLI/refs/heads/main/gfcli.sh
```

## Usage
simply run:
```bash
. gfcli.sh
```

## Credits
